About

WeatherApp provides current weather and short-term forecast using OpenWeather API.

Created by Bhaskar Charan for Portfolio project.

Technologies Used:

Frontend: HTML, CSS, JavaScript, and Bootstrap for responsive design and layout.

API Integration: OpenWeatherMap API to fetch real-time weather data, including temperature, humidity, wind speed, UV index, air quality, and sunrise/sunset times.

Extras: JavaScript Web Speech API for voice search and Geolocation API for automatic location detection.
