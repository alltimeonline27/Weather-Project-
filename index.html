About

WeatherApp provides current weather and short-term forecast using OpenWeather API.

Created by Team G for GMIT training project.

Technologies Used:

Frontend: HTML, CSS, JavaScript, and Bootstrap for responsive design and layout.

API Integration: OpenWeatherMap API to fetch real-time weather data, including temperature, humidity, wind speed, UV index, air quality, hourly-forecast, 5-Day forecast, and sunrise/sunset times.

Extras: JavaScript Web Speech API for voice search and Geolocation API for automatic location detection.

Data Storage: LocalStorage to save recent searches and user ratings for a personalized experience.

Our App Features:

  • Search weather by city name.
  • Voice input for city search.
  • Automatic location detection.
  • Display of current weather conditions.
  • Hourly and 5-day weather forecasts.
  • Air quality index information.
  • Recent searches saved for quick access.
  • User rating system for feedback.
  • Contact-form
  • Responsive View and User friendly interface
  • User login and logout system
  • Dark mood troggle
  • Background-color change according to weather information
  • Local time with date and sunrise & sunset time
  • Weather & health warning
